# AgentMint — iNFT Generator on 0G

Build an iNFT (intelligent NFT) generator for the 0G Zero Cup hackathon.

## Stack
- Solidity 0.8.24 + Hardhat (ERC-721 iNFT contract)
- Next.js 14 + wagmi v2 + RainbowKit
- 0G Storage (personality metadata)
- 0G Compute (personality generation + chat)
- 0G Chain Galileo testnet (chainId 16602)
